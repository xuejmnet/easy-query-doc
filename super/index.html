                                                                                                                                                                                                  # 超级功能

                                                                                                                                                                                                  easy-query的超级功能主要是分表分库和读写分离,作者有着多年的分库分表经验,并且在其他语言上也有一套相对完善的分表分库orm扩展
                                                                                                                                                                                                  (sharding-core)[https://github.com/dotnetcore/sharding-coreopen in new window]

                                                                                                                                                                                                  # 目录

                                                                                                                                                                                                  • 《分表》 数据库表进行分表处理
                                                                                                                                                                                                  • 《分库》 数据库对象分库处理

                                                                                                                                                                                                  所有的超级功能的案例源码都在easy-query-sampleopen in new window
